Ray Reflect

Download Github

Ray Reflect is a technical demo of shader-based raytraced reflections. It was made as part of my research and experimenting with both Vulkan and ray tracing. Lighting is calculated with a deferred renderer: the first pass renderes a G buffer with all the data needed for lighting calculations. The second step does those lighting calculations and casts a ray to determine reflections. Ray intersection is calculated as part of the fragment shader during the second pass.

3D Pool Simulation

Download Github

3D Pool Simulation was a primarily a physics demonstration. The physics engine simulates a set of spheres bouncing around a box, and a simple core OpenGL based renderer draws them as wireframes. Move the mouse around to aim, and click and hold left mouse to charge up and fire. I made all the physics, graphics, and sounds for the program.

Fist Fight Game

Download Github

Fist Fight is a game about fists and faces. There is only one goal: punch. There is only one rule: there are no rules.

Fist fight was made at as part of one of RPI Game Development Club's 24hr game jam. It was a bit of an experiment in multitasking. Each player controls both of their character's fists at once, one with each joystick.

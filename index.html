CatECS/Cat Engine

Download Github

Cat Engine is my own personal game engine. Its core is written in C, with some of the ECS systems being in C++. This demo demonstrates the engine's Entity Component System, scene loading, and prefabs. Scenes are loaded from specially formatted JSON files, and can be added to other scenes as prefabs. This demo uses a simple OpenGL sprite renderer.

While the demo is currently running in a single thread, the ecs supports multiple threads. The ECS is set up to minimize the amount of code that has to be thread-safe. The schedule is set up on engine starup, and systems are run on separate threads using a simple work-sharing approach. Each system can only write to the table column it is responsible for updating. The main thread is responsible for polling input and swapping the back buffer after all of the systems are finished for each frame.

Note: This will not compile under MSVC. It uses certain aspects of the C99 standard which MSVC does not support.

Ray Reflect

Download Github

Ray Reflect is a technical demo of shader-based raytraced reflections. It was made as part of my research and experimenting with both Vulkan and ray tracing. Lighting is calculated with a deferred renderer: the first pass renderes a G buffer with all the data needed for lighting calculations. The second step does those lighting calculations and casts a ray to determine reflections. Ray intersection is calculated as part of the fragment shader during the second pass.

3D Pool Simulation

Download Github

3D Pool Simulation was a primarily a physics demonstration. The physics engine simulates a set of spheres bouncing around a box, and a simple core OpenGL based renderer draws them as wireframes. Move the mouse around to aim, and click and hold left mouse to charge up and fire. I made all the physics, graphics, and sounds for the program.
